cc.factorie.infer

Maximize

trait Maximize[-A <: Iterable[variable.Var], -B <: model.Model] extends Infer[A, B]

An inference engine that finds score-maximizing values. The "infer" method returns a summary holding the maximizing assignment, but does not change the current variable values. By convention, subclass-implemented "apply" methods should change the current variable values to those that maximize; this convention differs from other Infer instances, which do not typically change variable values.
