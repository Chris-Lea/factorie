MutableCategoricalVar

trait MutableCategoricalVar[C] extends CategoricalVar[C] with MutableDiscreteVar

An abstract variable whose values are CategoricalValues, each corresponding to an integer 0...domain.size, and each associated with a category of type C (usually a String).
